🖥️ Applications using FluidSynth

Below is the list of applications that use FluidSynth as their audio engine.

Feel free to add your application to this page if it uses FluidSynth. For music or other audio created with FluidSynth, please see the MadeWithFluidSynth page.

  • aisings.com - automatic AI based music generation site that pulls influence from example music. (dead link)
  • ANMP - multichannel, loop-able video game music player for audiophiles and nerds
  • Ardour - DAW, comes bundled with a-fluidsynth LV2 plugin.
  • Audacious - Open source audio player for Linux, BSD derivatives and Microsoft Windows.
  • https://beatscratch.io - BeatScratch uses FluidSynth for playback on Android.
  • Beeano Midi - MIDI player and converter for Windows.
  • Cholidean Harmony Structure - A projection of 12-tone Equal Temperament music systems into 3D space that uses fluidsynth as MIDI player.
  • Cydrea Music - Cydrea Music combines A.I. and procedural generation to bring you a catalog of 65 quadrillion songs that you can play, edit, export (audio and MIDI) and use for free for content creation, game development, music production, etc. The free tier gives you 100+ songs every day, forever.
  • Denemo - front end for LilyPond music typesetter.
  • DOSBox Staging - x86/DOS emulator.
  • Drumstick MIDI Player - featured Multiplatform MIDI File Player with Karaoke support.
  • eplayOrgan - Windows, Mac and Linux multipurpose organ.
  • exeCute - Opens exe and bat files as if they were native Linux applications.
  • FluidGUI - graphical front end for FluidSynth by Ken Ellinwood.
  • FluidPatcher - A Python wrapper combining instrument settings, router rules, etc. into patches. Useful for performance or creating unique sound combinations.
  • FluidPi - Headless synth for Raspberry Pi. So minimal it works from console, controlled using a web server written in bash!
  • FluidSynth.clap - CLAP plugin for FluidSynth.
  • FluidSynth-DSSI - DSSI front end for FluidSynth.
  • FluidSynthGUI - windows software sampler based on FluidSynth engine. Select your midi input device, assign sound fonts to channels, and play!
  • FluidSynthPlugin - a VST3 plugin for a DAW and the pedantic FluidSynthFileConverter both using the FluidSynth library producing identical output (with a spectral difference of less than -100dBFS)
  • Gluid - mini FluidSynth front end for the system tray.
  • Hachiko - A tool creating pitch timeline from keystroke.
  • JJazzLab - A complete and open application for automatic backing tracks generation.
  • jOrgan - java virtual organ utilizing FluidSynth for pipe sound generation.
  • juicysfplugin - macOS AU/VST plugin + standalone app for rendering MIDI input through soundfont. JUCE framework.
  • ~KMid~ - ~featured KDE4 MIDI/karaoke player~, now deprecated and unmaintained. Replacement for Qt5/Qt6: Drumstick MIDI Player.
  • LilypondToBandVideoConverter: a generator for notation backing track videos from lilypond files orchestrating lilypond, fluidsynth, sox and ffmpeg.
  • LMMS - free and open-source cross-platform digital audio workstation.
  • Maestro - Midi Player Tool Kit - Free - FluidSynth on Unity
  • Maestro - Midi Player Tool Kit - Pro - FluidSynth with extra features on Unity
  • Mechanical Tunes - AI generated streaming music
  • MidiEditor - MIDI editor for Windows & Linux.
  • MidiGurdy - Electronic musical instrument (hurdy-gurdy) using FluidSynth as internal synthesizer
  • MonAMI - universal sensor framework, with a FluidSynth reporting plugin.
  • MusE - Linux Music Editor by Werner Schweer, a multi-track sequencer and audio application.
  • MuseScore - free music composition & notation software.
  • MuSIC-TeN Pianos - free virtual piano app on Microsoft app store that plays MIDI and MuSIC-TeN files.
  • mt32-pi - a baremetal kernel for the Raspberry Pi that turns it into a MIDI synthesizer, based on Munt and FluidSynth.
  • PianoBooster - MIDI File Player that teaches you how to play the piano.
  • Pocket Composer - Ful music theory app that help you to compose and create songs from scratch.
  • pygame - multimedia library for the Python language. Uses fluidsynth to play midi music, mostly on linux.
  • Radium - music editor inspired by the tracker interface.
  • Raagas - Powered by FluidSynth, the app renders generated MIDI into expressive, high-fidelity sound using real Indian instrument soundfonts, bringing algorithmic compositions to life on device.
  • QSynth - Qt front end for FluidSynth brought to you by Rui Capela.
  • Quote2Note - Translates stock quotes into musical notes.
  • ScoreDate - open source software written in Java that helps musicians to learn music reading and ear training.
  • ScummVM - program which allows you to run certain classic graphical point-and-click adventure games.
  • Simutrans - this game is a freeware and open-source transportation simulator. Your goal is to establish a successful transport company. Transport passengers, mail and goods by rail, road, ship, and even air. Interconnect districts, cities, public buildings, industries and tourist attractions by building a transport network.
  • SoundFont-Midi-Player - Android MIDI player app with fast switching of SoundFont files.
  • Swami - collection of software projects for editing and sharing instruments by Josh Green.
  • The Miditzer - computer program for PC's that lets you recreate a Wurlitzer Style 216 theater organ.
  • TuxGuitar - multi-track guitar tablature editor and player that supports many tablature file formats.
  • ViPiano - virtual MIDI Piano Keyboard with FluidSynth for Sailfish OS.
  • VLC - highly portable multimedia player for various audio and video formats.
  • ~VMPK & FluidSynth~ - ~virtual MIDI Piano Keyboard with FluidSynth for MeeGo Harmattan devices (Nokia N9 and N950)~. Deprecated and unmaintained.
  • VMPK - virtual MIDI Piano Keyboard with FluidSynth backend.
  • wxFluid - wxWidgets based FluidSynth frontend with RTP-MIDI and Network MIDI 2.0 support. Runs on Windows, macOS and Raspberry Pi OS
  • XunScore - A music notation software.
  • Z-Maestro - Windows application which aims to be similar to Apple Garageband by Zachary Northrup.
